🔥 Multi-Directional Shuttle Runs
~30 sec work
90 sec rest
5–6 reps
One rep: Sprint 5m → touch → back → sprint 10m → touch → back → sprint 15m → touch → back. Rest 90 sec. Repeat.
Why: Lactic system — the 10–45 sec bursts of repositioning, saving, crossing. Multi-directional mirrors match demands.
Ankle note: Even surfaces. Early weeks use 4m / 8m / 12m distances.
